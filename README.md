## About The Project

**게이트키퍼**는 수 많은 물리적 자물쇠 및 잠금키의 하드웨어 구조를 하나로 통일하고, 열린 시간 및 사용자의 정보를 확보하여 기존 잠금장치의 한계를 극복한 시스템을 개발하고자 하는 프로젝트입니다.
